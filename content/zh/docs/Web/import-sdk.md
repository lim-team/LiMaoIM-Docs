## 安装

#### NPM方式

狸猫IM支持 npm包引入方式 包名为limao

```Javascript
$ npm i limao
或者
$ yarn add limao
```

获得lim对象

```Javascript
import lim from 'limao'
```

#### script方式

进入狸猫IM [官网](http://limao.io)下载 JS SDK

```Javascript
 <script type='text/javascript' src="xxx/lim.js"></script>
```

获得lim对象

```Javascript
window.lim
```
